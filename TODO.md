# 待优化清单


## P0 — 高优先级（立即推进）

### 内容与体验
- [ ] 文章封面图（列表卡片展示，支持上传或外链）
- [ ] 文章目录 TOC（悬浮目录，H2/H3 自动生成，点击跳转 + 当前位置高亮）
- [ ] 代码块增强（语言标签 + 一键复制 + 行号显示）
- [ ] Toast 消息提示（替代 alert）
- [ ] 回到顶部按钮
- [ ] 评论嵌套回复（楼中楼，展示回复对象）

### SEO 与分发
- [ ] Meta 标签与 Open Graph（每篇文章动态生成 title / description / OG 标签）
- [ ] RSS 订阅（/rss.xml）
- [ ] Sitemap 自动生成（/sitemap.xml）
- [ ] HTTPS 与 HSTS（生产环境强制 HTTPS，配置 HSTS 头）

### 安全
- [ ] 登录失败次数限制（防暴力破解）
- [ ] 密码强度策略（最小长度、复杂度要求）
- [ ] 输入验证加强（所有 API 统一校验，拒绝无效输入）
- [ ] CSP 策略（Content-Security-Policy 头）

### 工程化
- [ ] 测试框架搭建（pytest + TestClient 覆盖后端 API）
- [ ] 数据库索引优化（slug、created_at 等常用查询字段加索引）
- [ ] GitHub Actions CI/CD（自动测试 → 构建镜像 → 部署）

---

## P1 — 中优先级（短期规划）

### 内容与体验
- [ ] 文章草稿功能（draft 状态，支持自动保存）
- [ ] 文章系列 / 专题（系列文章归类，自动显示系列目录和上/下篇导航）
- [ ] 文章置顶功能
- [ ] 文章列表多种排序（最新 / 最多点赞 / 最热）
- [ ] 相关文章推荐（基于标签匹配，文章底部推荐 3-5 篇）
- [ ] 阅读时间估算（根据字数估算，如"约 5 分钟阅读"）
- [ ] 阅读进度条（页面顶部细条显示阅读进度）
- [ ] Markdown 快捷工具栏（加粗、链接、图片插入按钮）
- [ ] 图片上传（拖拽或粘贴上传到图床）
- [ ] 图片灯箱（点击放大，ESC 关闭）
- [ ] 分享功能（复制链接 / 分享到 Twitter / 微信二维码）
- [ ] 访问量统计（文章阅读数 PV / UV）
- [ ] 评论列表分页加载

### 社交与互动
- [ ] 消息通知（评论回复提醒、系统通知）
- [ ] 收藏功能（收藏喜欢的文章）
- [ ] 邮件订阅（访客提交邮箱，新文章发布时邮件通知）

### 安全与管理
- [ ] 评论审核机制（新评论默认待审核，后台通过后展示）
- [ ] 垃圾评论过滤（敏感词 / 验证码）
- [ ] JWT 刷新令牌机制（支持令牌撤销）
- [ ] 操作日志记录（关键操作审计）
- [ ] CSRF 防护（CSRF token 验证）

### 工程化
- [ ] 前端测试（Vitest + Testing Library 组件测试）
- [ ] 代码质量工具（flake8 / black / isort 保证代码规范）
- [ ] 前端路由缓存（KeepAlive 保留列表滚动位置和筛选状态）
- [ ] 代码分割优化（按路由懒加载，减少首屏体积）
- [ ] 图片优化（自动压缩上传图片、WebP 格式）
- [ ] HTTP 缓存策略（合理设置缓存头，减少重复请求）
- [ ] PWA 支持（Service Worker + manifest，离线阅读、可安装）
- [ ] 健康检查端点（/health）
- [ ] 结构化日志（JSON 格式日志）

- [ ] 文章导出（Markdown / PDF）
- [ ] 文章批量操作（批量删除、批量分类）

- [ ] 评论表情支持（Emoji）
